## PROJECT SNAPSHOT (Initial Seed)

**Project**: NoKey Wallet
**Repository**: PolkaDotHackathon
**Purpose**: A Polkadot-powered proof-of-concept wallet that eliminates seed phrases using Web2-style authentication patterns, device-bound encryption, and social recovery.

**Stack**:
- Frontend: React 18.3.1 + Vite 5.4.11
- Styling: Tailwind CSS 3.4.15
- Animations: Framer Motion 11.11.17
- Crypto: @polkadot/util-crypto 13.2.3, crypto-js 4.2.0
- Icons: Lucide React 0.460.0
- Build: Vite with React plugin

**Architecture**:
- User Interface (React + Tailwind + Framer Motion)
  - Authentication Layer (Mock biometric, email verification, social recovery)
  - Key Management Simulation (Polkadot address gen, AES encryption, localStorage)
  - Mock Blockchain Layer (Simulated transactions, DOT balance, parachain connections)
  - UI Components (Landing, Dashboard, Modals)

**Key Features**:
1. No seed phrases - users never see private keys
2. Device-bound security simulation
3. Social recovery with trusted contacts
4. Email recovery flow
5. Parachain connections (Acala, Moonbeam, Astar, Parallel)
6. Transaction management (send/receive)
7. Modern Apple-style UI
